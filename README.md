## 使用指南

本 Games 系统主要目的是在 cmd 中直接运行。

OK，下面我们进行教学：

首先，打开 C 盘，找到 Windows 的文件夹，再找到 System32 的文件夹，把 Games 文件夹中的四个 .exe 文件拖进去，再打开 cmd 就可以了。

你没听错，就是这么简单。
